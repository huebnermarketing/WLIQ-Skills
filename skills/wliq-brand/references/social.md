# WLIQ Social Media Guidelines

## Post Specifications
| Spec | Value |
|---|---|
| Post size | 1080px × 1350px |
| Margin | 100px all sides |
| Logo height | 60px |

## Logo Placement Options
- Artboard Top Center
- Artboard Top Left (within margin)
- Artboard Bottom Left (within margin)
- Artboard Bottom Right (within margin)

## Light Post Styling
| Element | Value |
|---|---|
| Background | #FFFFFF |
| Title color | #000000 |
| Subtitle color | #777777 |
| Copy color | #333333 |
| Line art stroke | 2px, #333333 |

## Dark Post Styling
| Element | Value |
|---|---|
| Background | #000000 |
| Title color | #FFFFFF |
| Subtitle color | #777777 |
| Copy color | #ABABAB |
| Line art stroke | 2px, #777777 |

## Typography for Social
- **Title**: Avenir Black, large, all-caps or bold title case
- **SubTitle**: Avenir Heavy, #777777
- **Copy**: Avenir Medium or Roman, appropriate size for readability at 1080px

## Graphic Styles by Post Type

### Type 1 — One-line drawing post (LinkedIn & Instagram)
- Monochromatic line art, hand-drawn style
- Light bg: 2px #333333 strokes
- Dark bg: 2px #777777 strokes
- Minimal, clean — one concept per illustration

### Type 2 — Pastel illustration post (LinkedIn article cover)
- Full-color flat illustrations
- Pastel background colors (soft peach, pale blue, light yellow, mint, lavender)
- Characters are illustrated in simple flat style
- Used for article banners and feature posts

### Type 3 — Color highlight post (LinkedIn + Instagram)
- Base: one-line drawing style
- One element of the drawing is filled with a single pastel color
- Adds visual emphasis without breaking the monochromatic system

### Type 4 — Bold typography post (Instagram)
- Dark or black background
- Large Avenir Black headline, white
- Simple or minimal illustration as supporting element
- High contrast, bold messaging

## WLIQ Icon Fold Usage (Carousels)
For posts with **multiple slides**, add the WLIQ icon fold to the **bottom-right corner** of each slide.

| Version | Post Background | Fold Dimension | Fold Color |
|---|---|---|---|
| Light | #FFFFFF | 100px × 100px | #000000 |
| Dark | #000000 | 100px × 100px | #FFFFFF |

The fold is the IQ icon's characteristic corner tab — it visually "peeks" from the bottom-right, acting as both a brand mark and a carousel indicator.

## Platform Notes

### LinkedIn
- Article covers: pastel full-color illustrations
- Feed posts: bold typography + line art (light or dark bg)
- Carousel posts: use icon fold system

### Instagram
- Primarily dark posts with bold Avenir Black headlines
- One-line illustrations or color-highlighted versions
- Maintain the same 1080×1350px canvas
